The set of all #tex2html_wrap_inline19960# satisfying the #tex2html_wrap_inline19962# constraints is called the feasible region of the problem. The goal is to find a point #tex2html_wrap_inline19964# in the feasible region at which #math764##tex2html_wrap_inline19966# is an extremum.

In the most general case where the objective function #math765##tex2html_wrap_inline19968# and the constraint functions #math766##tex2html_wrap_inline19970# and #math767##tex2html_wrap_inline19972# are nonlinear, the process of solving this nonlinear optimization problem is called <#3829#>nonlinear programming (NLP)<#3829#>. Specially, if the objective function is quadratic while the constraints are linear, (the feasible region is a polytope), the process is called <#3830#>quadratic programming (QP)<#3830#>. More specially, if the objective function is linear as well as the constraints, the process is called <#3831#>linear programming (LP)<#3831#>.
